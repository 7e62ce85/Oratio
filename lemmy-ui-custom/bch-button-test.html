테스트 방법:

  • 자동 테마 감지: 브라우저 설정에서 다크 모드를 켜고 끄면서 첫 번째 버튼의 색상 변화를 확인하세요.
  • 호버 효과: 각 버튼에 마우스를 올려보세요.
  • 포커스 효과: Tab 키를 사용하여 버튼에 포커스를 맞춰보세요.
  • 접근성: 스크린 리더나 키보드 내비게이션을 테스트해보세요.
  • 고대비 모드: 브라우저의 고대비 모드에서 버튼이 잘 보이는지 확인하세요.

구현된 기능:

  • ✅ 브라우저 테마 자동 감지 (prefers-color-scheme)
  • ✅ 다크/라이트 모드에 최적화된 색상
  • ✅ 부드러운 호버 및 포커스 애니메이션
  • ✅ WCAG 가이드라인을 준수한 접근성
  • ✅ 고대비 모드 지원
  • ✅ 모션 감소 모드 지원 (prefers-reduced-motion)
  • ✅ 키보드 내비게이션 지원
  • ✅ 반응형 디자인
